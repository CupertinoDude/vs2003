Microsoft .NET Framework SDK Tool Developer's Documentation

This documentation provides developers with information useful for building low-level development tools such as compilers, browsers, profilers and debuggers.

Developer Services
Debug Reference
Provides a detailed description of the unmanaged debugging API.
Debugging Services
Provides an overview of debugging scenarios.
Profiling API
Provides details of the API used to monitor the performance and memory usage of an executing program.

Common Language Infrastructure (CLI)
The specifications in this section were submitted to ECMA for the standardization of the CLI.
Partition I Architecture
Provides an overview of the high level architecture of the entire common language infrastructure.
Partition II Metadata
Provides an overview of the Common Intermediate Language (CIL) and metadata.
Partition III CIL
Provides comprehensive reference documentation for the CIL instruction set.
Partition IV Library
Provides an overview of the CLI libraries and a specification of their factoring into Profiles and Libraries.
Partition V Annexes
Provides a collection of background information supporting the CLI standard.

Miscellaneous
Assembly Metadata Unmanaged API
Specifies the API for emitting and importing the portion of metadata that describes the assembly (the assembly manifest).
CIL Disassembler Advanced Options
Describes the advanced command line switches and features for use with the CIL Disassembler (Ildasm.exe)
Customer Debug Probes (CDP)
This feature can be enabled to help diagnose application problems and is particularly useful for isolating hard-to-find application bugs that occur when transitioning between managed and unmanaged code.
Declarative Security
Provides background information on .NET Framework security declarations and annotations that software development tools should support.
Hosting Interfaces
This specification describes the programming interfaces that runtime hosts call from unmanaged code to configure and load the CLR into a process.
Metadata API
Specifies the unmanaged API for emitting and importing metadata within an assembly (classes, methods, fields, and so on).
SOS
Provides an extended set of commands for the NT System Debugger (NTSD). This feature is useful for debugging the common language runtime at a very low level.
