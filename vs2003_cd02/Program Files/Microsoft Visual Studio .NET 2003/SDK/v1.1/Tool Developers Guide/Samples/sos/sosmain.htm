SOS - Son of Strike

SOS is an extension DLL that allows you to view internal CLR data structures from CDB and Windbg debuggers.  (It's a revised version of a previous tool called "Strike"; thus the name).  To use SOS follow these instructions

1. Install the 32-bit version of "Debugging Tools for Windows"

    This download is available from the Driver Development Kits Web site at: http://www.microsoft.com/ddk/debugging/installx86.asp
2. Extend the PATH enviroment variable to include Windbg and CDB tools.

3.  Extend the PATH enviroment variable to include the location of SOS.DLL at:

    %windir%\Microsoft.NET\Framework\<version>

4. Open a dump file with Windbg or CDB or attach to a process using the following commands:

    cdb -z <FULL file dump to path>

       or, for live debugging:

    cdb -pn aspnet_wp.exe

5. Load the SOS debug extension with the following command: 

	0:00> .load sos
