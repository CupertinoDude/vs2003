DumpStackObjects
Dumps all the objects used on the current thread
Syntax
!DumpStackObjects [<top stack>[<bottom stack>]]
Parameters
<top stack> Stack pointer at top of stack segment you wish to display
<bottom stack> Stack pointer at bottom of stack segment you wish to display
Example 
0:000> !DumpStackObjects

Comments

Use the DumpObj command with the object address to dump out the contents of the objects.

ESP/REG indicates the location of the object on the stack
