ObjSize
Finds the number of bytes that a root or all roots keep alive on the garbage-collected heap.
Syntax
!ObjSize [<addr>]
Parameters
<addr> Address of the object to get size of. If no address is given, all objects on all threads will be shown.
Example 
0:001> !ObjSize 010a50e0
sizeof(010a50e0) =       28 (    0x1c) bytes (Behavior1.Behavior1)
