GCRoot
Finds roots on the stack or handle for an object.
Syntax
!GCRoot <addr>
Parameters
<addr> Address of tha object to find handles for.
Example

010a50ec is the address of a System.Threading.ManualResetEvent found on the stack using the DumpStackObjects command.
