DumpDomain
Lists assemblies and modules in a domain
Syntax
!DumpDomain [<addr>]
Parameters
<addr>
domain address		 You can get the domain address of the thread from the Threads command. If you don't provide an address all current domains will be dumped

Comments

Shows the addresses of the heaps for the current domain and all assemblies loaded within the domain.
You can dump out the heaps with the DumpHeap command.
