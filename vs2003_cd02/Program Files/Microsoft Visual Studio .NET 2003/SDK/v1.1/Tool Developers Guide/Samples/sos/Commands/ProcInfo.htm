ProcInfo
Displays the process information.
Syntax
!ProcInfo [-env] [-time] [-mem]
Parameters
-env Displays environment variables.
-time Displays process time.  This parameter works only on live debugs.
-mem Displays memory usage.  This parameter works only on live debugs.
Example
