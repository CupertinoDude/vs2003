EEHeap
Lists information for the garbage-collected and Loader heaps
Syntax
!EEHeap [-gc] [-win32] [-loader]
Parameters
-gc Show garbage-collected heap
-win32 Show Win32 heap
-loader Show Loader heap
Example

Comments

For each domain and for the JIT code heap, the following information is shown: 
System Domain: 793b6e08
LowFrequencyHeap:00cd0000(2000:00001000) 
Size: 0x00001000(4096) bytes.
HighFrequencyHeap:00cd2000(8000:00001000) 
Size: 0x00001000(4096) bytes.
StubHeap:00cda000(2000) 
Size: 0x00002000(8192) bytes.
Total size: 0x4000(16384)bytes

00cd0000 is the virtual address of the heap
2000 is the virtual size (hex) of the heap
00001000 is the current size (hex) of the heap

Finally the total size of all heaps for all domains and the JIT code heap is summed up.
