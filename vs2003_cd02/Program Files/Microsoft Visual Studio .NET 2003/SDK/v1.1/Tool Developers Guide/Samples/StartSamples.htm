Microsoft .NET Framework SDK Tool Developer's Samples

Demonstrates how to develop compilers, debuggers, profilers and other developer and system services that support the Microsoft .NET Framework.

Exploring the samples

The Microsoft .NET Framework Tool Developer's samples are located in individual sub directories under the \Tools Developers Guide\Samples directory. To help you locate the sample you are looking for, you can view the sample readme files by clicking on the following links.

  ADepends Sample
  Views dependencies between assemblies.
  CLisp Compiler
  A managed Lisp compiler.
  CountDown ASM Application
  A large application written in the Microsoft Intermediate Language (MSIL) assembly language.
  Command Line Debugger
  Demonstrates the debugging APIs available in the common language runtime.
  MetaInfo - Metadata dump utility
  Dumps the metadata for a DLL or EXE file.
  MyC Compiler
  A simple compiler for a subset of the C language.
  Code Profiler
  Collects information about appdomains, assemblies, modules, classes, functions and function enter-leave events during the execution of a managed program.
  Common Language Runtime Profiler Hotspots Tracker
  Illustrates how to use the common language runtime Profiler API for identifying hotspots in managed code.
  Simple Managed C Compiler
  Compiler for an object oriented language similar to C++ that targets the common language runtime.

  Customer Debug Probes
  Customer Debug Probes (CDP) are switches in the runtime which will help you inspect runtime state which you would not otherwise be able to inspect and to receive notice of runtime events you could not otherwise trap.
