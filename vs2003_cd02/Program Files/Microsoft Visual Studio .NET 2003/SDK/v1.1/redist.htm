Microsoft .NET Framework Redistributable Package

The Microsoft .NET Framework version 1.1 is an integrated part of the Microsoft Windows Server 2003 family. The .NET Framework is also available to end users via Windows Update. In addition, it is available for redistribution to through the .NET Framework Redistributable Package.

See the .NET Framework Deployment Guide on MSDN for more information.

Microsoft Visual J# Redistributable Package

To run the Visual J# applications, including the Visual J# tools and samples, the Visual J# .NET Redistributable Package 1.1 must be installed. For more information, see http://msdn.microsoft.com/vjsharp.

If you run JBImp.exe, a Visual J# tool without installing the Visual J# .NET Redistributable Package 1.1, you will get the following error:

    Unhandled Exception: System.TypeInitializationException: The type initializer for "com.ms.vjsharp.cor.jbimp.BjCorGen" threw an exception. ---> System.IO.File Not FoundException: File or assembly name vjscor, or one of its dependencies, was not found.
You will also not be able to build and run Visual J# samples without installing Visual J# .NET Redistributable Package 1.1 first.
