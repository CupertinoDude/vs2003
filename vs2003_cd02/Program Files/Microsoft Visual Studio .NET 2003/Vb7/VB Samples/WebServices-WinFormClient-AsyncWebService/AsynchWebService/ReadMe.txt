Visual Basic .NET - Async Web Service

To install and run the Async Web Service sample:

1. Download and unzip the sample on your local system. 

2. Run the AsyncWebServiceSetup.msi and follow the directions.

3. After the installation completes, launch AsyncWebService.sln. 

	Note:  If VS.NET can't open the project via Front Page Server Extensions, a warning dialog will 
	       be displayed asking to switch to file based access.  Choose OK to continue opening the project

4.  Press F5 to run the sample.

5.  Click the Call Web Service Sync button.

6.  Click the Call Web Service Async button (Don't clear the 'Async Call Started' message to see the asynchronous behavior of the call).
