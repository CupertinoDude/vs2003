Microsoft Visual Studio .NET Prerequisites

Components with Languages Version Conflicts

Visual Studio .NET setup requires that certain prerequisites be installed in the same language version as the language version of your operating system. However, Visual Studio .NET setup does not include the language version that your operating system requires for these components.

You need to download the correct language version from the Internet, using the links below in section 1.1, "Required matching language versions". When you have installed all the components required for your operating system, you can return to setup by running Setup on Disk 1 of the Visual Studio product you are installing.

Also, you can install matching language versions for additional prerequisites using the links below in section 1.2, "Optional matching language versions". These components are not required to be installed in the same language version as the language version of your operating system. If you do not choose to install the matching language version, these components will be installed in the default language version, English, when you return to Visual Studio .NET setup.

Note It is recommended that you print or bookmark this page before installing the components listed below. This page does not automatically display after a machine restart.

1.1 Required matching language versions

Windows 2000 Service Pack
Windows 2000 Service Pack 3 is required. If subsequent service packs are available, download and install those as well. For more information, see http://www.microsoft.com/windows2000/downloads/servicepacks/default.asp

1.2 Optional matching language versions available

Microsoft Internet Explorer 6 Service Pack 1
Setup does not include a language version of Internet Explorer 6 Service Pack 1 that matches the language version of your operating system. You can, however, proceed with Visual Studio .NET setup and install the English language version. If you would like to install a matching language version, use the links below.

For languages other than Hebrew and Arabic, you can install the English version included with Visual Studio .NET. (Hebrew and Arabic versions require a setup specific to their respective languages.) To install the English version if you are installing from CD, run ie6setup.exe from the IE60SP1 folder on this CD; or, if you are installing from DVD, run ie6setup.exe from the WCU\IE60SP1 folder.

If you would rather install Internet Explorer in the language of your operating system, please see http://www.microsoft.com/Windows/IE/intl/default.asp for more information on downloading non-English versions.

Return to Setup

When you have installed all the components required for your operating system, you can return to setup by running Setup on Disk 1 of the Visual Studio product you are installing.
