The MyC Compiler

Welcome to the MyC page.  The primary goal of the MyC compiler is to show the implementation of various features of the Common Language Runtime. The only "language" design feature is that MyC intends to be a C-language subset.

Switches

/debug generate debug info
/nodebug do not generate debug info
/list generate an assembly listing (.asm)
/dll create a dll
/exe create an exe
/outdir:path create output files in this directory
