DirectX 7 Interoperability Sample – Using DirectMusic

This document contains the release information for the DirectX Interoperability Sample – Using DirectMusic.  You will find the following sections below:

  • Sample Overview
  • Requirements
  • Location of the Sample
  • Building the Sample
  • Running the Sample

 

Sample Overview

This sample demonstrates the use of an existing unmanaged COM component (DirectX 7) from managed code.  This sample contains one C# project that serves as a "jukebox", playing the tunes selected by the user. Although DirectX 7 is not the most recent version of DirectX, it is used since Windows 2000 already has it installed. 

 

Requirements

This samples requires DirectX 7 (which is already installed with Windows 2000) to function properly.

 

Location of the Sample

This sample is located in the Interop\Basic\DirectX directory of the .NET Framework SDK samples directory.

For Example:

C:\Program Files\Microsoft.NET\SDK\v1.1\Samples\Technologies\Interop\Basic\DirectX\DirectMusic

  PlayStyles subdir – Demonstrates playing the various DirectMusic musical styles.

 

Building the Sample

In order to properly build this sample, the following steps need to be performed prior to running the sample:

·        Type nmake all from the command line.

For Example:

Nmake all

 

Running the Sample

In order to run this sample, follow the instructions listed below.

  • Open the "PlayStyles.exe" executable in the PlayStyles sub-directory.
  • Click on the desired style.
