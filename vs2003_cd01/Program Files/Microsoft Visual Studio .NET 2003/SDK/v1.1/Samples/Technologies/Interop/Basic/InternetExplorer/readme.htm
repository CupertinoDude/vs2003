Internet Explorer Event Interoperability Sample

This document contains the release information for the Internet Explorer Event Interoperability sample.  You will find the following sections below:

 

  • Sample Overview
  • Location of the sample
  • Building the sample
  • Running the sample

 

Sample Overview

This sample demonstrates automating Internet Explorer from managed code and sinking Internet Explorer events in managed code.

 

Location of the Sample

This sample is located in the Interop\Basic\InternetExplorer directory of the .NET Framework SDK samples directory.

For Example:

C:\Program Files\Microsoft.NET\SDK\v1.1\Samples\Technologies\Interop\Basic\InternetExplorer

 

Building the Sample

In order to properly build this sample, the following step needs to be performed:

Type NMAKE from the command line.

For Example:

NMAKE

 

Running the Sample

In order to run this sample, follow the instructions listed below.

·         Run Explorer from the command line after building the sample successfully.

For Example:

Explorer

·         Type in the URL of a website on the command line to go to that website.

·         To quit, press enter.
