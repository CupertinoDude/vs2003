Active Directory Interoperability Sample

This document contains the release information for the Active Directory Interoperability sample.  You will find the following sections below:

 

  • Sample Overview
  • Requirements
  • Location of the sample
  • Building the sample
  • Running the sample

 

Sample Overview

This sample demonstrates using Active Directory service from managed code.     

Requirements

This sample requires Win2K or later.

Location of the Sample         

This sample list located in the ...\Interop\Basic\ActiveDS directory of the .NET Framework SDK sample directories.

For Example:

C:\Program Files\Microsoft.NET\SDK\v1.1\Samples\Technologies\Interop\Basic\ActiveDS

 

Building the Sample

In order to properly build and run this sample, the following steps need to be performed prior to building and running the sample:

·         Type nmake all from the command line.

For Example:

Nmake all

 

Running the Sample

This sample can only be built and run on Win2K or later.  In order to run this sample, follow the instructions listed below.

·         Build the project by typing nmake from the command line in the ActiveDS directory.

·         Type AdsComputer from the command line.

For Example:

nmake

AdsComputer
