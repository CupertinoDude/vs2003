ManagedEvents Interoperability Sample

This document contains the release information for the ManagedEvents Interoperability sample.  You will find the following sections below:

 

  • Sample Overview
  • Location of the sample
  • Building the sample
  • Running the sample

 

Sample Overview

This sample demonstrates how a VB6 client sinks events fired from managed code.    

Location of the Sample         

This sample is located in the Interop\Basic\ManagedEvents directory of the .NET Framework SDK samples directory.

For Example:

C:\Program Files\Microsoft.NET\SDK\v1.1\Samples\Technologies\Interop\Basic\ManagedEvents

 

Building the Sample

In order to properly build this sample, the following step needs to be performed:

Type NMAKE from the command line.

For Example:

NMAKE

Running the Sample

In order to run this sample, follow the instructions listed below.

·         Build the project by typing nmake from the command line in the ManagedEvents directory.

·         Copy sink.exe from VBSink directory to ManagedEvents directory.

·         Type Sink from the command line.

For Example:

nmake all

copy .\VBSink\Sink.exe

Sink
