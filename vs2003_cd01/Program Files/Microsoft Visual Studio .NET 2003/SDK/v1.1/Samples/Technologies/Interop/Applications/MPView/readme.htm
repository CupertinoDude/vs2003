MPView Interoperability Sample

This document contains the release information for the IEnumVariant - IEnumerator Interoperability sample.  You will find the following sections below:

 

  • Sample Overview
  • Requirements
  • Location of the Sample
  • Building the sample
  • Running the sample

 

Sample Overview

This sample demonstrates bridging of IEnumVariant from COM and IEnumerator from managed code.  

 

Requirements

This sample requires Windows 2000 or later and the windows.h header file.

 

Location of the Sample         

This sample is located in the Interop\Applications\MPView directory of the .NET Framework SDK samples directory.

For Example:

C:\Program Files\Microsoft.NET\SDK\v1.1\Samples\Technologies\Interop\Applications\MPView

 

Building the Sample

In order to properly build this sample, the following step needs to be performed:

Type NMAKE from the command line.

For Example:

NMAKE

Running the Sample

In order to run this sample, follow the instructions listed below.

·         Build the project by typing nmake from the command line in the MPView directory.

·         Type MPView from the command line.

For Example:

Nmake

MPView
