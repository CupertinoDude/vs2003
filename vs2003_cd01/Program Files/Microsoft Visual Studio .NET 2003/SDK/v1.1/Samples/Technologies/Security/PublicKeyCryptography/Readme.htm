Overview

This sample demonstrates how public key encryption can be used to securely exchange messages between 3 entities.  The messages are encrypted using a symmetric algorithm (RC2) and then the RC2 key is encrypted using an asymmetric algorithm (RSA). Asymmetric cryptographic algorithms, also known as public-key algorithms, require that each entity maintains a pair of related keys: a private key and a public key. Both keys are unique to the entity. The public key can be made available to anyone; this key is used for encoding data to be sent to a receiver. The private key must be kept private by the receiver; this key is used for decoding messages encoded using the receiver's public key.

Topics

The following bullets briefly describe the classes and technologies used by this sample.

  • Security.Cryptography -
    • RSACryptoServiceProvider - The .NET implementation of RSA, which is an asymmetric (public key) algorithm. RSA is used to encrypt the RC2 key to be sent with the encrypted message.
    • RC2CryptoServiceProvider - A symmetric encryption algorithm which is used to encrypt the message text.
    • ICryptoTransform - This interface defines the basic operations of cryptographic transformations.
    • CryptoStream - A stream used to encrypt data which is fed into a memorystream.
  • I/O -
    • MemoryStream - The memory stream is used to temporarily store the output of encryption.
  • Text -
    • Encoding - The Encoding.Unicode class is used to convert strings to byte arrays and vice versa.

Location

This sample is located in the Technologies\Security\PublicKeyCryptography subdirectory of the .NET Framework SDK samples directory.

For Example:

C:\Program Files\Microsoft.NET\SDK\v1.1\Samples\Technologies\Security\PublicKeyCryptography

Building the Sample

In order to properly build and run this sample, the following steps need to be performed.

Type BUILD.bat from the command line.

For Example:

C:\Program Files\Microsoft.NET\SDK\v1.1\Samples\Technologies\Security\PublicKeyCryptography\VB>Build.bat

Running the Sample

In order to run this sample, follow the instructions listed below.

  • This sample requires the Windows 2000 High Encryption Pack.  Additional information and downloads of the high encryption pack are available on the Microsoft web site:

    http://www.microsoft.com/WINDOWS2000/downloads/recommended/encryption/
  • Type PublicKey.exe from the command line

    For Example:

    C:\Program Files\Microsoft.NET\SDK\v1.1\Samples\Technologies\Security\PublicKeyCryptography\VB>PublicKey.exe
