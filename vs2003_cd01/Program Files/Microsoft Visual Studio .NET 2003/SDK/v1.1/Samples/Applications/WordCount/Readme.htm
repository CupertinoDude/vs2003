Overview

This sample offers a nice introduction to the base class library. It shows how to create an application that opens multiple files (specified on the command line) and counts each file’s bytes, characters, words, and lines. The results from each file and the total of all files are displayed.

Topics

The following bullets briefly describe the classes and technologies used by this sample.

  • I/O -
    • FileStream - The sample uses this type for its file access. This class is used to read to and write from files.
    • StreamWriter - The sample uses the StreamWriter when the command-line arguments request that the output be sent to a file. The StreamWriter is used in conjunction with a FileStream instance to format text output to a file.
    • StreamReader - The sample uses the StreamReader to read the contents of a file for which it is counting words. The StreamReader is used in conjunction with a FileStream instance to read text from a file.
  • Collections -
    • ArrayList - Used by the sample as a general collection class for storing sets of objects.
    • SortedList - Used by the sample as a collection class for storing sets of objects in a sorted manner.
    • IEnumerator - This sample uses this interface directly to enumerate over sets of objects.
    • IDictionaryEnumerator - This sample uses this interface directly to enumerate over indexed sets of objects.

Location

This sample is located in the Applications\WordCount subdirectory of the .NET Framework SDK samples directory.

For Example:

C:\Program Files\Microsoft.NET\SDK\v1.1\Samples\Applications\WordCount

Building the Sample

In order to properly build and run this sample, the following steps need to be performed.

Type BUILD.bat from the command line.

For Example:

C:\Program Files\Microsoft.NET\SDK\v1.1\Samples\Applications\WordCount\VB>Build.bat

Running the Sample

In order to run this sample, follow the instructions listed below.

  • Type WordCount.exe from the command line for a complete list of command line options for the utility.

    For Example:

    C:\Program Files\Microsoft.NET\SDK\v1.1\Samples\Applications\WordCount\VB>WordCount.exe

  • To count the words in a file, type WordCount.exe [filename] from the command line.

    For Example:

    C:\Program Files\Microsoft.NET\SDK\v1.1\Samples\Applications\WordCount\VB>WordCount.exe ArgParser.vb
